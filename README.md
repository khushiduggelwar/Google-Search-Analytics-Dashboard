# Google-Search-Analytics-Dashboard

## Project Overview
Built an interactive Power BI dashboard to analyze Google Search Console and Analytics data across multiple projects, enabling performance tracking and decision-making.

---

## Business Problem
How to monitor website performance and identify trends in clicks, impressions, CTR, and user engagement across multiple projects?

---

## Key Features
- Created a master dataset by combining multiple project sheets
- Developed KPIs: Total Clicks, Impressions, CTR, Active Users, Sessions
- Added filters: Project Name, Date, Assignee, Project Type
- Built interactive visuals for daily and monthly performance tracking

---

## Tools & Technologies
- Excel (Data Cleaning)
- Power BI (Dashboard, DAX)
- Google Sheets (Data Source)

---

## Key Insights
- Identified top-performing projects based on clicks and impressions
- Observed CTR fluctuations across different dates
- Analyzed user engagement and session trends
